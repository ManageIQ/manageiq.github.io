Community

Get Involved

ManageIQ is open source and open for contributions. There are many ways to get involved, and everyone is invited to connect, discuss, develop and collaborate in the ManageIQ community!

Connect

Find us in your favorite social channel so you can be in the loop with announcements and happenings.

  • Follow @manageiq @manageiq on Twitter
  • ManageIQ on Facebook
  • ManageIQ on LinkedIn
  • Sprint Review slides
  • Event & Sprint Review videos

Collaborate

Access the source code and developer guides to get going. Refer to the above engagement channels if you need some assistance!

  • Main ManageIQ GitHub repo Star Fork
  • Members in ManageIQ community on GitHub
  • All repos under ManageIQ Follow @manageiq
  • Developer Guides

In addition, you don't have to be a developer to be able to contribute. Filing issues/bugs is a great way to help. The ManageIQ project also welcomes translations!

Discuss

Join the active ManageIQ community on ManageIQ Talk forum with your questions and discussions, or chat with us anytime on Gitter (sign in with your GitHub account).


Check out the ManageIQ blog and comment on the latest community news.

Community Events

Browse the Events Calendar and learn where you can meet up with other members of the community.
