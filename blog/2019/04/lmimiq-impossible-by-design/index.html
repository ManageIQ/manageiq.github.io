Last Month in ManageIQ: Playtest Session 1

Apr 30, 2019 by d-m-u
  • LWIMIQ

PLAYTEST SESSION 1 V4.30

Major Changes

  • Added formalized Nightmares
  • Redefined Starvation
  • Clarified Lunch

Added traits:

  • Restlessness
  • Indolence
  • Factory for the embedded ansible cloud credential
  • Clumsiness

Altered traits:

  • Moved GIT_REPO_DIRECTORY constant from MiqAeDatastore
  • Made it easier to induce comas
  • Fixed incorrect key sent back for dynamic, nonsorted items
  • Increased hallucinations with found health potions if exhaustion levels > 2
  • Used string class name instead of hitting autoload
  • Updated rails to 5.0.7.2
  • Made load_values_on_init for dialog fields independent of show_refresh_button
  • Correctly set Conversion host base class
  • Codified reactions of second level post-humans in immersive social networks
  • Allowed non-root users to run miqssh as themselves
  • Lost HP for use of precognition, false empathy or the mortal sin of mixing fractions with percents
  • Made remote console worker dependencies load optionally
  • Added zone to service provisioning

Removed traits:

  • Removed references to MiqStats
