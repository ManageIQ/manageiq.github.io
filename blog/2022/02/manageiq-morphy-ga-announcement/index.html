ManageIQ Morphy GA Announcement

Feb 14, 2022 by chessbyte
  • releases
  • announcements

On behalf of the ManageIQ team, I’m happy to announce the release of ManageIQ Morphy! This release is named after the chess grandmaster Paul Morphy. You can download the Morphy-1 release on our downloads page.

Here are the highlights of the Morphy release:

  • Providers
    • Create IBM PowerVC Provider [#21098]
    • IBM Cloud VPC Provider Enhancements [#146]
    • IBM Cloud PowerVS Provider Enhancements [#269]
    • Support Oracle Cloud [#2]
    • Kyndryl AutoSDE Provider Enhancements [#82]
    • Support Oracle Kubernetes Engine (OKE) [#31]
    • Support Azure Kubernetes Service (AKS) [#442]
    • Support Google Kubernetes Engine (GKE) [#180]
    • Support IBM Cloud Kubernetes Service (IKS) [#149]
    • Add support for Cloud Databases [#20221]
    • Pull in support for vSphere 7.0U3 [#759]
  • Core
    • [EmbeddedAnsible] Configurable verify_ssl for git repositories [#20732]
    • Gracefully handle non-superuser setting up replication [#21372]
    • Faster migrations [#21259]
  • API
    • API Interface Changes [#1067]
    • Throttle API requests [#20955]
  • UI
    • Automation/Configuration menu reorganization [#7657]
    • Add provider connection status errors to provider dashboard [#7709]
    • Forms Conversion to React [#7989]
  • Build
    • [Podified] Upgrades: handle updates to CRs with new CRD settings [#676]
    • Move from CentOS 8 to CentOS Stream [#459]
    • Dependent containers should store a manifest of the git SHA and date [#699]
  • Developer
    • Support running logical replication locally for easier development [#21299]
    • Make provider secrets pluggable [#21157]

There are many more changes that would be impossible to detail in a single blog post. Many thanks goes to all of the community members for their contributions! We are already well underway on the next release of ManageIQ - Najdorf. If you’re interested in getting involved, check out our roadmap, check out our community page, ask questions in our talk forum, or chat with us in Gitter. We look forward to hearing from you!
