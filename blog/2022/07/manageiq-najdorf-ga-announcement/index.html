ManageIQ Najdorf GA Announcement

Jul 18, 2022 by Fryguy
  • releases
  • announcements

On behalf of the ManageIQ team, I’m happy to announce the release of ManageIQ Najdorf! This release is named after the chess grandmaster Miguel Najdorf. You can download the Najdorf-1 release on our downloads page.

Here are the highlights of the Najdorf release:

  • New Providers
    • IBM Power HMC provider [#21475]
    • IBM Cloud Infrastructure Center (CIC) provider [#21631]
    • Cisco Intersight provider [#21715]
    • VMware Tanzu Kubernetes Grid (TKG) provider [#703]
  • Provider Enhancements
    • IBM Cloud [#263]
      • Expanded network inventory collection (public gateways, VPN gateways, load balancers, ACLs, security group rules)
      • Support to modify/resize VM [#300]
      • Metrics collection [#308]
      • Catalog service provisioning [#362]
      • Create/Delete cloud databases [#364], [#366]
      • Add metrics collection support for IKS [#314]
    • IBM Power HMC [#8]
      • Expanded inventory collection (templates, storages)
      • Expanded network inventory collection (networks, virtual switches, virtual lans, ethernet adapters, logical ports)
      • Metrics collection [#41]
      • Targeted refresh [#692]
      • Template provisioning [#26]
    • IBM Power Virtual Servers [#147]
      • IBM Cloud ObjectStorage provider [#159]
      • PowerVC -> PowerVS cross cloud-provider image import [#180]
      • Event collection [#279]
      • Expanded inventory collection for snapshots [#313]
      • Snapshot operations [#325]
    • IBM PowerVC [#27]
      • Expanded inventory collection for advanced settings [#28]
      • OOtB reports [#36]
    • Kyndryl AutoSDE [#83]
      • Targeted refresh [#107]
      • Support for fibre channel WWPN candidates [#92]
      • Add Host Initiator Groups [#93]
    • VMware
      • Add SSL options for Verify / CA certificates [#737]
  • Core
    • Cross cloud-provider image import [#180]
    • Allow the addition of standalone object storage managers [#7675]
  • API
    • Localization of OPTIONS calls [#1043]
    • Add endpoints for cloud_subnets [#1025]
    • Add LAN tags subcollction [#1018]
  • Build
    • [Podified] Allow SSL/TLS communication between pods [#724]
    • Run the application as a non root user [#20394]
    • Update containers to latest ubi8 [#21725]
  • Major Bugs
    • Error out orphaned miqtask executed by processes/pods that no longer exist [#21175]
  • Developer
    • [UI] Update to NodeJS 14 [#7930]
    • [UI] Replace patternfly-react with carbon-react [#7844]
    • [UI] Continued work on removal of angular code [#8299]
    • [UI] Continued work on converting forms to react [#8010]
    • [UI] Continued work on converting HAML tables to React Carbon tables [#7986]
    • Replace Hakiri with Whitesource [#21660]
  • Removals
    • Remove v2v plugin [#21379]
    • Remove cockpit console support [#21378]
    • Remove database backups from UI (moved to appliance console) [#21100]

There are many more changes that would be impossible to detail in a single blog post. Many thanks goes to all of the community members for their contributions! We are already well underway on the next release of ManageIQ - Oparin. If you’re interested in getting involved, check out our roadmap, check out our community page, ask questions in our discussions forum, or chat with us in Gitter. We look forward to hearing from you!
