ManageIQ Radjabov GA Announcement

Apr 15, 2025 by Fryguy
  • releases
  • announcements

On behalf of the ManageIQ team, I’m happy to announce the release of ManageIQ Radjabov! This release is named after the chess grandmaster Teimour Radjabov. You can download the Radjabov-1 release on our downloads page.

Here are the highlights of the Radjabov release:

  • New Providers
    • Embedded Terraform
    • OpenShift Virtualization as a dedicated provider apart from KubeVirt
  • Providers
    • Cisco Intersight
      • Add support for Intersight Private Virtual Appliance [#103]
    • KubeVirt (and OpenShift Virtualization)
      • KubeVirt VM Metrics Capture [#260]
      • KubeVirt Events Capture [#256]
      • Service Catalog Provisioning Support [#250]
      • Restart a Virtual Machine [#263]
      • Create snapshot for a Virtual Machine [#267]
  • Core
    • Added new product feature that controls who can see Request Logs [#22701]
    • Upgrade Ruby to 3.3 [#23142]
    • Upgrade Rails to 7.0 [#22052]
    • Workflows
      • Add provisioning to workflows [#23039]
      • Use Floe wait for event driven updates [#109]
      • Expose requester to workflows [#101]
      • Add an embedded_ansible builtin method [#86]
      • Add provision_execute builtin method [#81]
      • Rename ‘builtin’ scheme to ‘manageiq’ [#96]
      • Provide out of the box workflows [#744], [#750]
      • [ASL] Add Intrinsic Functions [floe#64]
      • [ASL] Implement Map state [floe#184]
      • [ASL] Implement Parallel State [floe#287]
  • UI
    • Many page conversions from Angular to React
    • Many page conversions from HAML to React [#9396], [#9399], [#9400]
  • API
    • Add /api/providers/#/metric_rollups [#22827]
  • Build
    • [Appliances] Upgrade to EL9 [#563]
    • [Appliances] Set Ruby stream to 3.3 [#578]
    • [Containerized] Upgrade to UBI9 [#848]
    • [Operator] Upgrade the operator to UBI9 [#1055]
  • Developer
    • Update Yarn to v4.3.1 [#23072]
    • Default PostgreSQL encryption to scram-sha-256 [#388]

IMPORTANT: The following are potentially breaking changes and deprecations

  • **BREAKING**
    • CentOS Stream 8 is no longer supported.
    • Removal of PostgreSQL 10 [#23065]
    • Removal of Ruby 3.0 support, Ruby 3.1 is the minimum

There are many more changes and bug fixes that would be impossible to detail in a single blog post, but you can read through them all in the full changelog. Many thanks goes to all of the community members for their contributions!

We are already well underway on the next release of ManageIQ: Spassky. If you’re interested in getting involved, check out our roadmap, check out our community page, ask questions in our discussions forum, and chat with us in Gitter. We look forward to hearing from you!
